# harkirat-chahal.github.io — personal site

Simple static portfolio. No build step, no frameworks — just HTML + CSS.

## Deploy on GitHub Pages (10 minutes)

1. Create a GitHub account if you don't have one (pick a clean username — it goes on your resume).
2. Create a new **public** repo named exactly `YOUR-USERNAME.github.io`
   (e.g. `harkiratchahal.github.io`).
3. Upload these files to the repo (drag-and-drop works on github.com: "Add file" → "Upload files"):
   - `index.html`
   - `style.css`
   - `Harkirat_Chahal_Resume.pdf` (export your final resume as PDF and drop it in)
4. Wait ~2 minutes. Your site is live at `https://YOUR-USERNAME.github.io`.

## Before going live — find & replace

- Replace every `YOUR-USERNAME` in `index.html` with your GitHub username.
- Replace each `YOUR-USERNAME/REPO` project link with the real repo URL
  (push your OMMS fork/branch, job board, and WebGL code to public repos —
  or remove the "source" link for anything you can't make public, like a
  private capstone repo).
- Add the resume PDF, or remove the two resume links.

## Editing later

Edit `index.html` directly on github.com and commit — the site updates
automatically in a minute or two.
